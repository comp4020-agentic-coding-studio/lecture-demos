# lecture-demos

The live demo repo for the COMP4020/COMP8020 Agentic Coding Studio lectures
(Semester 2, 2026, ANU). Every lecture's demo happens here: one repo, worked in
week by week for the whole semester.

It's empty until teaching starts in Week 1 (27 July 2026). From then on:

- the demo is pushed at the end of each beat, so `git pull` during a lecture
  puts you on the exact state showing on the projector
- the end of each week is tagged (`week-01` through `week-12`)
- the history never gets cleaned up --- the commit log is the honest record of
  what the agent (and the convenor) actually did

To follow along: clone this repo in Week 1, then start each lecture with
`git fetch && git reset --hard origin/main`. Keep anything you want to hold onto
from the "now you try" pauses on your own branch.
